A Deforestation B Agriculture C Burning of fossil fuels D Urbanization

A Dunes B Mangroves C Rocky cliffs D Forests

A Jet Stream B Thermohaline circulation C El Niño-Southern Oscillation D Trade Winds

A La Niña B Monsoon C Heatwave D El Niño

A Central Asia B Sub-Saharan Africa C Southeast Asia D Northern Europe

A Tundra B Mangroves C Grasslands D Deserts

A Ecosystem-based adaptation B Structural adaptation C Non-structural adaptation D Behavioral adaptation

A Sedimentation B Erosion C Liquefaction D Subsidence

A Sustainability B Resilience C Adaptation D Mitigation
